Web Programlama

Nedir?

Web programlama biraz kafa karıştırıcı, alışagelmedik bir tabir olabilir. Çoğu kimsenin web tasarım olarak bildiği bu ifade, dinamik sitelerin internet üzerinde hakimiyeti ele geçirmesiyle niteliğini kazanmıştır. Statik, değiştirilmesi zor bir Web tasarımı, yeniliklerin ve güncellemelerin zor yapıldığı broşür niteliğindeki web siteleri tercih edilmemektedir. Doğru projelendirme, iyi bir analiz ile sitenizi tamamen sizlerin güncelleyeceği, sizlerle iş yapacak insanların, sizler hakkında doğru bilgileri elde etmesini sağlayacak web tasarımına, web programlama denir. Web programlama, bir web programlama dili ve veritabanı ile dinamik içerikli web siteleri yapmayı amaçlar. Bu yolla yapılan siteler, statik görsellikten uzak, güncel ve işlevseldirler. Alışveriş siteleri, haber siteleri, forum siteleri ve bunlar gibi birçok site web programlamayla oluşturulur. Web Programlama, temel ve basit anlamıyla bir web sitesi oluşturmak için yazılan kaynak koddur. Web Programlama temelde 2 ye ayrılır. İstemci (Front-End) ve Sunucu (Back-End) taraflı programlamadır.
